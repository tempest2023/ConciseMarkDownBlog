# My First Blog Post

Welcome to your new blog! This sample post shows you how to write blog posts using Markdown.

## Headings

Use `#` for headings. More `#` symbols mean smaller headings:

# Heading 1
## Heading 2
### Heading 3

## Text Formatting

You can make text **bold**, *italic*, or ~~strikethrough~~.

## Links and Images

Links: [Visit GitHub](https://github.com)

Images:
```markdown
![Alt text](https://via.placeholder.com/300x200)
```

## Code Blocks

Inline code: `console.log('Hello!')`

Code block with syntax highlighting:

```javascript
function greet(name) {
  return `Hello, ${name}!`;
}

console.log(greet('World'));
```

```python
def greet(name):
    return f"Hello, {name}!"

print(greet("World"))
```
